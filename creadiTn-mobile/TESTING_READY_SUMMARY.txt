## 🐛 Troubleshooting

| Problem | Solution |
|---------|----------|
| Backend not responding | Check: `netstat -ano \| findstr 8081` |
| API connection failed | Update API_BASE_URL in `src/api/config.ts` |
| Image upload fails | Ensure `uploads/` directory writable |
| Slow startup | Run `npm run build` and reload |
| Database errors | Verify MySQL running and created database |
